# Simple Weather App

A beautiful and responsive weather application where users can search for any location and view detailed weather information.

## Features

- **Location Search**: Type any city or place name to get weather data
- **Real-time Weather**: Displays current temperature, conditions, and feels like temperature
- **Detailed Information**:
  - Current temperature
  - Feels like temperature
  - Humidity percentage
  - Wind speed
  - Atmospheric pressure
  - Visibility distance
  - UV index
  - Weather condition with emoji icons

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Beautiful UI**: Modern gradient design with smooth animations
- **Free API**: Uses Open-Meteo API (no API key required)

## How to Use

1. **Open the App**: Open `index.html` in any modern web browser
2. **Search for Weather**: 
   - Type a city or location name in the search box
   - Press Enter or click the Search button
   - Weather information will display instantly

3. **Try Different Locations**: 
   - Search for "London"
   - Search for "Tokyo"
   - Search for "Paris"
   - Any city name works!
